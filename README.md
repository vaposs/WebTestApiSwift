# Приложение для тестирования веб работы приложения на swift.
## Что сделано:

### Cерверная часть для получение запросов от клиента и запись данных в базу данных на Postgres

## Что нужно для работы:
 - создать базу данных
 - изменить параметры доступа в базу данных (скрипт configure)
 - изменения параметров user - строчка 15
 - изменения пароля доступа в БД - строчка 16
 - изменения названия БД - строчка 16
   
## Запуск и проверка работы: 
   - собрать проект - swift build
должны увидеть строчку в терминале - [3081/3081] Applying WebTestApi Build of product 'WebTestApi' complete! (107.08s)
   - запустить проект - swift run 
должны увидеть строчку в терминале - [ NOTICE ] Server started on http://127.0.0.1:8080 
   - открыть вторую вкладку терминала и проверить доступность базы данных - psql <имя базы данных> 
   - во строй вкладке выполнить команду проверки созданных таблиц - \dt (должны увидеть users и homes) 
   - запустить файл в папке Public - WebTestApi - должно открыться браузер с тестовой страницей 
   - выполнить несколько раз добавление информации. 
   - выполнить во втором окне терминала команды - SELECT * FROM users; (или homes) для проверки записей в Базе данных
   - все)
